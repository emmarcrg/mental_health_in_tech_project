# mental_health_in_tech_project

En ce basant sur le dataset Mental Health in Tech Survey, je vais essayer de définir quels sont les éléments les plus importants pour définir la santé mentale d'un employé, voir définir les personnes les plus à risques.

## Problèmatique : 

Quelles sont les caractéristiques qui peuvent augmenter le risque de maladie mentale chez un individu ? 

Est-il possible de définir la probabilité pour qu'une personne ait une maladie mentale ?
